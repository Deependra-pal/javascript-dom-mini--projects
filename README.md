# javascript-dom-mini--projects
 Javascript DOM mini project for practice

This repository contains beginner - friendly Javascript DOM mini project built using HTML , CSS , Javascript.

## Project Included 
  - counter-app
  - Theme Toggle (Dark / Light Mode)
    
## Counter App 
A simple counter application where users can :-
 - Increase the count
 - Decrease the count
 - Reset the count

This project is built to practice:-
-DOM manipulation
-Event handling
-Javascript logic with UI

## Purpose
The purpuse of this repository is to practice Javascript DOM concepts and built a strong foundation for frontend  development.
